# Apple Notes

## ショートカット

### メモを書き始める

| アクション | ショートカット |
| --- | --- |
| メモを新規作成する | Command + N |
| クイックメモを開く | Fn + Q (Command + Tabで戻る) |
| 選択中のメモを開く | Return (ESCで戻る) |

### ウィンドウ

| アクション | ショートカット |
| --- | --- |
| メインウィンドウに移動する | Command + 0 |

### テーブル

| アクション | ショートカット |
| --- | --- |
| テーブルを追加する | Option + Command + T |
| 現在の列の右（左）に列を追加する | Option + Command + 右矢印キー（左矢印キー） |
| 現在の行の下（上）に行を追加する | Option + Command + 下矢印キー（上矢印キー） |
